# 02_Dependencias Principales

## Core

- **`expo`**: Framework principal SDK 50+.
- **`react-native`**: Motor de renderizado móvil.
- **`expo-router`**: Gestión de navegación basada en archivos.

## UI & Estilos

- **`expo-linear-gradient`**: Fondos degradados.
- **`@expo/vector-icons`**: Iconos (FontAwesome, Material).
- **`expo-haptics`**: Feedback táctil (vibraciones).

## Datos & Mapas

- **`expo-sqlite`**: Base de datos local persistente.
- **`react-native-maps`**: Integración de mapas (Google/Apple Maps).
- **`expo-location`**: Geolocalización del dispositivo.

## Gestión de Estado

- **React Context API**: Manejo de estado global (Carrito, Favoritos).
- **React Hooks**: lógica encapsulada en Controladores.
